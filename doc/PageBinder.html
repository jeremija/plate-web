Class: PageBinder

PageBinder

This object handles the `page-route-found` event and coordinates the page show/hide method calls and page route changes. It also listens to `page-route-not-found` event and redirects to the error page.

Hiddes the current page (if any). If the new page is already bound, it will be shown and it's route callback applied. If the new page is not yet bound to an element, it's template will be loaded.
