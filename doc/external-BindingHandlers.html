BindingHandlers

Knockout's binding handlers. The BindingHandlers are documented as functions, even though they are infact objects. The paramters described show the parameters which can be used in the `data-bind` attribute in the HTML.

invalidFields(invalidFields)

Reads the object with invalid fields properties and adds the `.has-error` class to fields which have the same name. Also adds `bs.tooltip` to the input item. Use this binding on a `form` element.
Parameters:
Name Type Description
invalidFields Observable A map of invalid fields as keys. Values are a external:ValidationError objects.
